# Cut, Copy and Paste in JavaScript with the Clipboard API

Browser Clipboard API demonstrations by [Craig Buckler](https://twitter.com/craigbuckler) for our [article on programmatically accessing the clipboard using JavaScript](https://www.sitepoint.com/clipboard-api/).

## Requirements

A modern browser.

## Installation Steps 

None. Just open any of the HTML files in the root of the project.
